## Обзор проекта
Этот проект является **универсальным стартовым шаблоном (boilerplate)** для разработки расширений Visual Studio Code на TypeScript. Он содержит готовые примеры реализации наиболее востребованных функций VS Code API.

### Реализованные возможности (Boilerplate Features)
- **Commands**: Регистрация команд в Command Palette.
- **Settings**: Управление конфигурацией расширения (чтение и прослушивание изменений).
- **Activity Bar & Side Bar**: Кастомная иконка на боковой панели и дерево (TreeView).
- **Status Bar Item**: Интерактивный элемент в нижней панели редактора.
- **Webview API**: Создание полноценных HTML-интерфейсов.
- **Background Jobs**: Фоновые задачи с отображением прогресса (Progress API) и работой с файловой системой.
- **Context Menus**: Интеграция в меню проводника (Explorer).

### Архитектура
- `src/extension.ts`: Главная точка входа. Содержит примеры активации всех API.
- `esbuild.js`: Скрипт сборки. Включает плагин для работы с `problemMatcher` в VS Code, что позволяет отладке по `F5` запускаться мгновенно.
- `package.json`: Конфигурация всех "Contribution Points".
- `media/`: Ресурсы (иконки и т.д.).

## Сборка и запуск

### Разработка
- **Установка зависимостей:** `npm install`
- **Компиляция:** `npm run compile`
- **Режим наблюдения:** `npm run watch` (автоматическая пересборка при изменениях).
- **Отладка:** Нажмите `F5` в VS Code. Откроется окно `[Extension Development Host]`, где расширение будет активно.

### Проверка типов и линтинг
- **Линтинг:** `npm run lint`
- **Проверка типов:** `npm run check-types`

## Инструкции для использования как шаблона
1. Измените `publisher` в `package.json`.
2. Замените иконку в `media/icon.svg` на свою.
3. Удалите ненужные блоки в `src/extension.ts` (каждый блок пронумерован и снабжен комментарием).
4. Соответствующим образом очистите секцию `contributes` в `package.json`.
